# Gym Management System for a Gym Owner in Morocco

As a **gym owner in Morocco**, I would want my management system to do one main thing:

> Help me make more money, lose fewer members, and control everything easily from phone and desktop.

The system should not be only an admin tool. It should be a complete **business platform** that manages members, payments, coaches, renewals, marketing, and growth.

---

## 1. Member Management

The system must contain a complete member profile.

| Feature             | Why It Matters                               |
| ------------------- | -------------------------------------------- |
| Member profile      | Name, phone, email, photo, emergency contact |
| CIN optional        | Useful for identification if needed          |
| Subscription status | Active, expired, frozen, trial, VIP          |
| Start and end date  | Avoid confusion about expiry                 |
| Membership history  | See all past subscriptions                   |
| Notes               | Injuries, preferences, special cases         |

The owner and receptionist should be able to search members quickly by:

* Name
* Phone number
* QR code
* Badge/card
* Membership ID

---

## 2. Subscriptions and Pricing

The system should support real Moroccan gym offers.

| Subscription Type         | Example                          |
| ------------------------- | -------------------------------- |
| Monthly subscription      | 250–400 MAD                      |
| 3-month package           | Discounted offer                 |
| 6-month package           | Medium-term commitment           |
| Yearly subscription       | Best-value package               |
| Personal training package | 8, 12, or 20 sessions            |
| Group classes package     | Zumba, boxing, yoga, HIIT        |
| Frozen subscription       | Travel, sickness, Ramadan, exams |

The system should also support:

* Discounts
* Promo codes
* Free trial days
* Referral offers
* Renewal reminders
* Custom pricing
* Special Ramadan offers
* Corporate/company packages

---

## 3. Payments and Invoices

The payment module should be strong because money control is the heart of the business.

| Feature            | Details                                            |
| ------------------ | -------------------------------------------------- |
| Cash payment       | Very important in Morocco                          |
| Bank transfer      | Mark manually or upload proof                      |
| Partial payment    | Example: client pays 200 MAD now and 100 MAD later |
| Debt tracking      | Know who still owes money                          |
| Receipts           | PDF or WhatsApp receipt                            |
| Daily cash closing | Control receptionist cash                          |
| Revenue dashboard  | Daily, weekly, monthly income                      |

Payment categories should include:

* Memberships
* Personal training
* Group classes
* Products
* Events
* Penalties
* Other income

---

## 4. Renewal and Email Reminders

The system should send or prepare messages for:

| Reminder Type            | Example                                         |
| ------------------------ | ----------------------------------------------- |
| Expiry reminder          | “Votre abonnement expire dans 3 jours.”         |
| Renewal message          | “Renouvelez aujourd’hui et gardez votre tarif.” |
| Abandoned lead follow-up | For people who asked for price but did not join |
| Birthday message         | Loyalty and personal touch                      |
| Promotion message        | Ramadan offer, summer offer, student offer      |

Important feature:

> One click to send a reminder or message from the member profile.

---

## 5. Staff and Coach Management

As a gym owner, I need to control my team.

| Feature               | Details                                    |
| --------------------- | ------------------------------------------ |
| Staff roles           | Owner, manager, receptionist, coach        |
| Permissions           | Receptionist cannot delete payments        |
| Coach schedule        | Personal training sessions and classes     |
| Commission tracking   | Coach earns commission per session/package |
| Staff attendance      | Check-in and check-out                     |
| Performance dashboard | Know which coach brings most revenue       |

This helps avoid problems like:

* Who accepted this payment?
* Who gave this discount?
* Who deleted this member?
* Which coach generated the most income?

---

## 6. Lead and Sales CRM

A good gym system should not only manage existing members. It should help bring new ones.

| Feature                | Why It Matters                                          |
| ---------------------- | ------------------------------------------------------- |
| Lead database          | Store people who ask on Instagram, WhatsApp, or walk in |
| Source tracking        | Instagram, Facebook, walk-in, referral                  |
| Sales status           | New, contacted, visited, converted, lost                |
| Follow-up reminders    | Remind staff to call or message                         |
| Trial session tracking | Know who tried but did not subscribe                    |
| Conversion dashboard   | See which channel brings paying clients                 |

Example statuses:

* New lead
* Contacted
* Interested
* Trial booked
* Trial completed
* Converted
* Lost

---

## 7. Products and Stock

Many gyms sell products, so the system should include a mini POS module.

| Feature         | Examples                              |
| --------------- | ------------------------------------- |
| Product stock   | Protein, water, gloves, belts, towels |
| Barcode scan    | Fast product sale                     |
| Low-stock alert | “Only 3 protein boxes left”           |
| Profit tracking | Buy price vs sell price               |
| Sales by staff  | Know who sold what                    |

---

## 8. Equipment and Maintenance

A serious gym needs equipment tracking.

| Feature               | Details                              |
| --------------------- | ------------------------------------ |
| Equipment list        | Treadmills, bikes, machines, weights |
| Maintenance dates     | Last and next maintenance date       |
| Broken machine status | Out of service                       |
| Repair cost           | Track expenses                       |
| Supplier contact      | Machine supplier info                |

This helps protect the customer experience and avoid broken equipment staying ignored.

---

## 9. Reports and Dashboards

As a gym owner, I want the dashboard to answer important business questions.

| Question                            | Dashboard Answer     |
| ----------------------------------- | -------------------- |
| How much money did we make today?   | Daily revenue        |
| How many active members do we have? | Active subscriptions |
| How many members expired this week? | Renewal risk         |
| Who owes money?                     | Debt list            |
| Which offer sells best?             | Package performance  |
| Which coach performs best?          | Coach revenue        |
| What are peak hours?                | Attendance heatmap   |
| Are we growing?                     | Monthly comparison   |

Important KPIs:

* Active members
* Monthly recurring revenue
* Renewal rate
* Churn rate
* Average revenue per member
* Unpaid amount
* New leads
* Conversion rate
* Best-selling subscription
---

The owner should control the whole business from the settings.

| Setting                | Examples                                     |
| ---------------------- | -------------------------------------------- |
| Gym branches           | Single branch or multiple locations          |
| Opening hours          | Normal schedule and Ramadan schedule         |
| Price plans            | Monthly, yearly, student, VIP                |
| Tax/invoice info       | Company details                              |
| Staff permissions      | Who can see money, edit members, delete data |
| Notification templates | French, Arabic, Darija                       |
| Payment methods        | Cash, card, transfer, online payment         |
